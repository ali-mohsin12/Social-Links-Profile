## 🎯 What I Learned

In this project, I practiced building a responsive card layout using Flexbox. I improved my understanding of alignment, spacing, and structuring content clearly.

I also learned how to create reusable CSS classes instead of repeating styles and how to style interactive elements like links with hover effects.

## 🚧 Challenges

I faced challenges with aligning elements correctly and making the layout consistent. I overcame this by using Flexbox and adjusting spacing step by step until the design matched closely.
